
## Develop ndm

Setup

`$ git clone https://github.com/720kb/ndm.git`

`$ cd ndm`

`$ npm install`

Run app

#### Run on Linux
`$ npm run linux`

#### Run on Mac
`$ npm run mac`

#### Run on Windows
`$ npm run win`


if you then want to test the executables follow [How to build](https://github.com/720kb/ndm/blob/master/doc/BUILD.md)
