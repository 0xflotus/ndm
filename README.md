###Develop

_Setup_

`$ git clone <repo> && cd ndm`

`$ npm install`

_Run app_

`$ npm start`


###Build

If you want to make your own executable:

`$ git clone <repo> && cd ndm`

`$ npm install`

`$ npm run build`

Executables will be output in `ndm/releases/` directory
