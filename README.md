## Homebrew

On MacOS you can install **ndm** with [Homebrew Cask](https://caskroom.github.io/):

```bash
$ brew update
$ brew cask install ndm
```

## Arch Linux

On Linux you can install **ndm** like this:

```bash
$ yaourt -S ndm
```
